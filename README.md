# Password Strength Checker - DecodeLabs Project 1

## What It Does
A Python program that checks password strength based on length, uppercase/lowercase letters, numbers, special characters, and common patterns.

## Features
- Evaluates password strength (Weak/Medium/Strong)
- Provides detailed feedback
- Checks for common passwords
- Runs in continuous loop

## How to Run
1. Run: `python password_checker.py`
2. Enter a password
3. View the strength result and feedback
4. Type 'quit' to exit
